# material-snap-on

# npm、できればyarnがインストールされている必要あり

## Project setup
node_modulesがインストールされます。npmでもyarnでもOK。
```
npm install
```
```
yarn install
```


### Compiles and hot-reloads for development
開発用に実行。localhost:8080で見られます。ホットリロードされる。
```
npm run serve
```
```
yarn serve
```

### Compiles and minifies for production
```
npm run build
```
